# gostop-chip-tracker

3~4명이 같은 시작 칩으로 게임을 시작하고, 매 판 끝난 뒤 플레이어 간 칩 이동만 기록하는 React 프론트엔드 앱입니다.

## 기능
- 3~4명 플레이어 관리
- 동일 시작 칩 설정
- 플레이어 간 칩 이동 기록
- 현재 잔액 자동 계산
- 최근 이동 내역 확인
- localStorage 저장

## 실행
```bash
npm install
npm run dev
```

## 주의
- 아직 테스트/빌드는 돌리지 않았습니다.
- 백엔드 없이 브라우저 localStorage에 저장합니다.
